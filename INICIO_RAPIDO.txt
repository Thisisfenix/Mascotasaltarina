⚡ INICIO RÁPIDO
═══════════════════════════════════════════════════════════════════════════════

┌─────────────────────────────────────────────────────────────────────────────┐
│ OPCIÓN 1: PYTHON (Multiplataforma)                                         │
└─────────────────────────────────────────────────────────────────────────────┘

  1. Instalar Python 3.8+ desde python.org
  2. Abrir terminal en la carpeta del proyecto
  3. Ejecutar:
     
     pip install -r requirements.txt
     python mascota_saltarina.py

┌─────────────────────────────────────────────────────────────────────────────┐
│ OPCIÓN 2: C# (Solo Windows, Mejor Rendimiento)                             │
└─────────────────────────────────────────────────────────────────────────────┘

  1. Instalar .NET 8.0+ desde dotnet.microsoft.com
  2. Abrir terminal en MascotaSaltarinaCS
  3. Ejecutar:
     
     cd MascotaSaltarinaCS
     dotnet run


🔨 COMPILAR EJECUTABLES
═══════════════════════════════════════════════════════════════════════════════

┌─────────────────────────────────────────────────────────────────────────────┐
│ PYTHON → EXE                                                                │
└─────────────────────────────────────────────────────────────────────────────┘

  pip install pyinstaller
  pyinstaller --onefile --windowed --icon=icon.ico mascota_saltarina.py
  
  → Ejecutable en: dist/mascota_saltarina.exe (~50-100 MB)

┌─────────────────────────────────────────────────────────────────────────────┐
│ C# → EXE (Versión Normal)                                                   │
└─────────────────────────────────────────────────────────────────────────────┘

  cd MascotaSaltarinaCS
  compilar.bat
  
  → Ejecutable en: publish/AnkushCat.exe (~15 MB)
  ⚠️  Requiere .NET instalado en el PC donde se ejecute

┌─────────────────────────────────────────────────────────────────────────────┐
│ C# → EXE (Versión Standalone)                                               │
└─────────────────────────────────────────────────────────────────────────────┘

  cd MascotaSaltarinaCS
  compilar_standalone.bat
  
  → Ejecutable en: publish-standalone/AnkushCat.exe (~60-80 MB)
  ✅ NO requiere .NET instalado, funciona en cualquier PC
