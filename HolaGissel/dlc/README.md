# 🎮 DLC: Pack de Juegos

## Descripción
Añade 5 nuevos minijuegos clásicos a Ankush Cat.

## Juegos Incluidos

### 🐍 Snake
- Controla la serpiente con las flechas
- Come la comida roja para crecer
- No choques con las paredes ni contigo mismo

### 🏓 Pong
- Usa las flechas arriba/abajo para mover tu paleta
- Juega contra la IA
- Primer jugador en anotar gana

### 🐱 Flappy Cat
- Presiona ESPACIO para saltar
- Evita los obstáculos verdes
- Acumula puntos pasando entre ellos

### 🎨 Simon Says
- Memoriza la secuencia de colores
- Repite la secuencia haciendo clic
- Cada nivel añade un color más

### 🔨 Whack-a-Mole
- Golpea los topos que aparecen
- Tienes 30 segundos
- Acumula la mayor cantidad de puntos

## Instalación
1. Coloca `pack_juegos.py` en la carpeta `dlc/`
2. Reinicia Ankush Cat
3. Ve a la pestaña "Juegos"
4. Haz clic en "🎮 Pack de Juegos DLC"
